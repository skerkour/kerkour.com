# Cross-platform shellcode in Rust

## Setup

```shell
$ make docker
```

## Usage


### x86_64

```shell
$ make hello_world_x86_64
$ make dump_hello_world_x86_64
$ make execute_x86_64
```

### Aarch64

```shell
$ make hello_world_aarch64
$ make dump_hello_world_aarch64
$ make execute_aarch64
```


## Clean

```shell
$ make clean
```
